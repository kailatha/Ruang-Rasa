# Neutral / Netral

## Gambaran Umum
Neutral pada konteks journaling menunjukkan user sedang berada dalam kondisi emosional yang relatif stabil, datar, atau sekadar mendeskripsikan kejadian tanpa emosi dominan.

Kondisi ini tidak berarti user tidak memiliki emosi.

## Bahasa yang Aman
Gunakan:
- "cukup tenang"
- "netral"
- "belum terasa dominan"
- "masih mencoba memahami perasaan"

Hindari:
- "kamu tidak punya emosi"
- "hidupmu membosankan"
- "kamu dingin"

## Jika Intensitas Ringan
Respons:
- bantu user mengeksplorasi pengalaman
- dorong refleksi ringan

Aktivitas:
- journaling harian
- refleksi aktivitas hari ini
- mindfulness singkat

## Jika Intensitas Sedang
Respons:
- bantu user mengenali pola kecil
- fokus pada kesadaran diri

Aktivitas:
- tracking suasana hati
- refleksi kebutuhan pribadi
- mindful observation

## Jika Intensitas Berat atau Sangat Berat
Respons:
- tetap validasi kondisi user
- jangan memaksa eksplorasi emosi mendalam

Aktivitas:
- grounding ringan
- observasi diri tanpa tekanan
