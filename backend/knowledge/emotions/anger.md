# Anger / Marah

## Gambaran Umum
Anger pada konteks journaling menunjukkan user sedang merasa kesal, marah, frustrasi, tersinggung, atau merasa diperlakukan tidak adil.

Label ini tidak boleh langsung dianggap sebagai perilaku agresif atau berbahaya.

## Bahasa yang Aman
Gunakan:
- "kesal"
- "frustrasi"
- "emosinya sedang tinggi"
- "marah terhadap situasi"
- "terasa tidak adil"

Hindari:
- "kamu terlalu emosional"
- "marah itu salah"
- "sudah lupakan saja"
- "kamu agresif"

## Jika Intensitas Ringan
Respons:
- validasi rasa frustrasi
- bantu user menamai emosi
- arahkan refleksi singkat

Aktivitas:
- journaling emosi
- tarik napas perlahan
- jeda sebelum bereaksi

## Jika Intensitas Sedang
Respons:
- bantu user menenangkan intensitas emosi
- hindari menyuruh langsung tenang
- fokus pada kebutuhan atau pemicu

Aktivitas:
- grounding
- mindfulness singkat
- problem-solving sederhana

## Jika Intensitas Berat atau Sangat Berat
Respons:
- gunakan bahasa netral dan tidak memicu
- jangan memperdebatkan emosi user
- prioritaskan stabilisasi emosi

Aktivitas:
- breathing exercise
- menjauh sementara dari pemicu
- hubungi orang aman bila perlu
